Tiny Layout

Tiny Layout is a tool based on HTML's native Web Components designed to make it easy to create interactive diagrams for education, based on SVG. Using a custom domain-specific language, you can easily define shapes and functions to make it easy to create diagrams quickly. Since it is built with Web Components, it can be used with any web framework, or without one at all.

Demo

Here's a quick demo that showcases the power of the language. You can write in the Tiny Layout language in the left pane, and see it be quickly compiled and run in the pane on the right, or play with the examples to see what's possible.

Concepts

Tiny Layout is intentionally small, so that it is easy to learn, and difficult to forget. To help you get started, there are a couple concepts that will be helpful.

  1. The only compound data type in Tiny Layout is table, which works similarly to maps and dictionaries in other languages. Since we don't have arrays, we use tables in their place, using indices starting with 0 as keys.
  2. All shapes in Tiny Layout are made from the small set of primatives: path, curve, ellipse, group, and text. If you want to use these to make more familiar shapes, you can define a set of functions that construct those using these primatives. For instance, a circle can be created from an ellipse, and a rectangle from a path.

Usage

After installing Tiny Layout, you can use it in your projects using the <tiny-layout> component. Simply write your Tiny Layout code between the opening and closing tags.

Reference

path: [points, fill?, stroke?, strokeWidth?, onClick?]

Draws a (possibly closed) sequence of straight lines. This is a great choice for creating both lines and polygons.

ellipse: [center, radius, fill?, stroke?, strokeWidth?, onClick?]

Draws an ellipse. This is a great choice for creating circles.

curve: [points, fill?, stroke?, strokeWidth?, onClick?]

Draws a curve using a quadratic bezier curve among every three points.

group: [body, fill?, stroke?, strokeWidth?, rotate?, onClick?]

Groups multiple elements under one container. Using fill, stroke and strokeWeight, you can set the default fill and stroke settings for all nested elements. Using rotate, you can rotatet the whole group without impacting the functions' view of the canvas.

text: [content, position, fill?, stroke?, strokeWidth?, fontSize?, fontFamily?, onClick?]

Draws text on the canvas.
