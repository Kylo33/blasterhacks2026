Tiny Layout

Tiny Layout is a tool designed to make it easy to create interactive diagrams for education, based on SVG. Using a custom domain-specific language, you can easily define shapes and functions to make it easy to create diagrams quickly.

Types

To be comprehensive, here is a list of all possible types in Tiny Layout, along with a description of what they represent. For fun, each type is exactly five characters, which is why some of them have silly names!

  1. numba represents Tiny Layout's number type; this includes both integers and floating point numbers. Numbers must be specified in decimal; there is no support for hex, octal, or binary.
  2. truth is Tiny Layout's boolean type, which can take on the values true and false.
  3. strin is the string type, which has support for the most common escape characters.
  4. table is used as Tiny Layout's hash table type. As noted in the concepts section, this is the only argument that can be passed to a function, and it is also used to represent more complicated data types.
  5. funct is Tiny Layout's function type, which can only be created as an anonymous function.
  6. zilch is Tiny Layout's void, or nil type.
  7. color is Tiny Layout's color, or nil type. It can be specified using either three or six hexadecimal digits (either the format #012 or #012345). Note that no quotes are used for the color type.
